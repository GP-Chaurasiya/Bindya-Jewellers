# Bindya Jewellers

A jewellery store website built with HTML, CSS, and JavaScript.

## Getting Started

```bash
git clone https://github.com/your-username/bindya-jewellers.git
cd bindya-jewellers
open index.html
```

## Features

- Image slider
- Login & Register popup
- Product collections (rings, necklaces, earrings, chains, bracelets)
- New arrivals section
- Responsive design
- Scroll to top button

## Tech Stack

HTML · CSS · JavaScript
